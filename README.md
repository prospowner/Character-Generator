# C++ Game Character Generator

A streamlined console utility written in C++ designed to demonstrate foundational Object-Oriented Programming (OOP) paradigms through explicit class blueprints, custom attributes, and dynamic instance instantiation matrices.

## Features
* **Explicit Class Blueprints:** Leverages encapsulated object modeling structures (`class Character`) featuring public member access specifiers to expose internal fields securely.
* **Instance State Mapping:** Instantiates distinct memory structures (`player1`, `player2`) loaded with individual identity strings and integer-mapped power arrays.
* **Inline Behavior Execution:** Features localized class-bound member methods (`greet()`) to stream live instance fields into standard text outputs flawlessly.

## How to Compile & Run
Ensure you have a C++ compiler installed (like GCC/MinGW or Clang). 

Compile the source architecture using your terminal tools:
```bash
g++ main.cpp -o generator
